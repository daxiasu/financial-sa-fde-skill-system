# 风险规则

## 风险维度

| 维度 | 典型关注点 |
|---|---|
| 主体资质 | 成立时间较短、法定代表人或股东频繁变更、经营异常、行政处罚、经营范围与授信用途不匹配 |
| 司法与舆情 | 被执行、失信、重大诉讼、行政处罚、负面新闻、安全环保事故、欠薪、停工 |
| 财务质量 | 收入下滑、利润质量弱、经营性现金流为负、应收账款高、存货高、资产负债率高 |
| 经营真实性 | 订单证据不足、客户集中、供应商集中、纳税/发票/物流/流水与收入不匹配 |
| 关联关系 | 关联交易、关联担保、交叉持股、同地址同电话、共同管理人员、资金回流 |
| 行业与区域 | 行业周期下行、政策收紧、区域产业风险、原材料或商品价格大幅波动 |
| 授信用途 | 用途笼统、与经营规模不匹配、存在资金挪用可能、存在借新还旧迹象 |
| 抵押担保 | 权属不清、重复抵押、估值依据不足、变现能力弱、担保链条复杂 |

## 建议权重

该分数是“走访前风险关注分”，分数越高表示客户经理在走访前需要投入越高关注度。

| 维度 | 权重 |
|---|---:|
| 主体资质与工商异常 | 15 |
| 司法舆情与行政处罚 | 20 |
| 财务质量与偿债压力 | 25 |
| 经营真实性 | 15 |
| 关联关系与资金闭环风险 | 15 |
| 行业环境与授信用途 | 10 |

## 等级映射

| 分数 | 等级 | 含义 |
|---:|---|---|
| 0-20 | L1 | 已提供资料中暂未发现明显风险信号 |
| 21-40 | L2 | 存在资料缺口或轻微信号 |
| 41-60 | L3 | 存在需要重点现场核验的风险线索 |
| 61-80 | L4 | 存在多项异常或较高风险 |
| 81-100 | L5 | 存在重大风险信号，需要人工专项复核 |

## 典型规则示例

### 财务质量

- 经营性现金流连续两年为负：属于重要风险信号。
- 应收账款占收入比例超过 40%：需要重点核验回款真实性和账期。
- 收入快速增长，但利润和现金流未同步改善：需要核验收入质量。
- 短期借款明显增加：需要核验偿债压力、再融资压力和真实资金缺口。

### 经营真实性

- 前五大客户收入占比超过 60%：需要关注客户集中度。
- 大额订单缺少合同、发票、物流、验收或回款凭证：需要核验交易真实性。
- 收入增长与纳税、开票、物流、银行流水等证据不匹配：需要重点交叉验证。

### 关联关系

- 实控人控制的贸易公司与申请企业存在交易：需要核验关联交易定价和资金流向。
- 存在较大关联担保或互保：需要核验或有负债。
- 多家公司存在同地址、同电话或共同管理人员：需要核验实际控制关系。

### 司法与舆情

- 存在失信或较大金额被执行记录：高关注。
- 存在多起供应商合同纠纷：需要核验付款压力和隐藏债务。
- 存在停工、欠薪、环保、安全、质量等负面舆情：需要现场核验并收集佐证材料。

## 等级选择指引

- L1 仅适用于资料相对完整，且未发现明显风险信号的情况。
- L2 适用于资料不足但暂未提供明确负面信号的情况。
- L3 适用于存在一个或多个需要现场重点核验的风险信号，例如高应收、现金流为负、客户集中、关联交易等。
- L4 适用于多个维度同时异常，例如股东频繁变更、现金流为负、诉讼较多等。
- L5 适用于重大风险信号，例如失信记录、大额被执行、重大欺诈嫌疑、严重负面舆情或经营真实性无法核验。

## 证据纪律

每一条风险线索必须标注其依据类型：

- 已提供证据
- 根据已提供数据推导
- 公开来源待核验
- 银行内部数据待核验
- 资料缺失

不得在证据不足时写出确定性结论。
