# 测试样例

## 测试样例 A：资料较完整，存在应收账款和关联交易风险

### 输入

- 企业名称：宁波某智能装备有限公司
- 所属行业：高端装备制造
- 所在地区：浙江省宁波市
- 授信类型：续贷
- 申请金额：3000 万元
- 授信用途：采购原材料和日常流动资金
- 已知背景：企业为某核心厂商二级供应商，近两年收入增长较快，但应收账款余额同步上升。
- 工商信息：企业成立 8 年，注册资本 5000 万元，实控人持股 65%，近一年无经营异常记录。
- 财务信息：近一年营业收入增长 35%，净利润增长 8%，经营性现金流连续两年为负，应收账款占营收比例 48%，资产负债率 62%。
- 经营信息：主要客户集中于 3 家大型制造企业，前五大客户收入占比约 70%。企业表示订单充足，但回款周期较长。
- 司法信息：无失信被执行记录，无重大未决诉讼。
- 舆情信息：暂无明显负面舆情。
- 关联关系：实控人另控制一家贸易公司，该贸易公司与本企业存在采购往来。
- 银行内部信息：历史还款正常，无逾期记录。

### 预期表现

- 走访前风险关注等级建议为 L3。
- 关键风险线索应包括经营性现金流为负、高应收账款、客户集中、关联交易。
- 现场问题应覆盖回款周期、核心客户稳定性、坏账计提、关联交易定价和现金流缺口。
- 补充材料应包括主要客户合同、发票、回款凭证、银行流水和关联交易说明。

## 测试样例 B：资料不足

### 输入

- 企业名称：成都某供应链服务有限公司
- 所属行业：供应链服务
- 所在地区：四川省成都市
- 授信类型：首次授信
- 申请金额：1000 万元
- 授信用途：补充流动资金
- 已知背景：客户由园区推荐，暂未提供详细经营资料。
- 其他资料：暂无

### 预期表现

- 不应直接判断企业为低风险或高风险。
- 等级建议为 L2，或说明“因资料不足暂不评分”。
- 报告应列出缺失的工商、财务、经营、司法、关联关系、银行流水等资料。
- 现场问题应聚焦基本商业模式、真实贸易背景、主要客户供应商和资金用途。

## 测试样例 C：存在重大司法和经营异常

### 输入

- 企业名称：武汉某建筑工程有限公司
- 所属行业：建筑工程
- 所在地区：湖北省武汉市
- 授信类型：增额
- 申请金额：5000 万元
- 授信用途：项目回款前的工程款垫付和日常周转
- 工商信息：近一年法定代表人和股东发生多次变更。
- 司法信息：存在多起买卖合同纠纷，被执行金额合计 1200 万元。
- 舆情信息：公开报道提及工程停工和拖欠供应商款项。
- 财务信息：短期借款大幅增加，经营性现金流为负。
- 银行内部信息：存量授信尚未逾期。

### 预期表现

- 走访前风险关注等级建议为 L4 或 L5。
- 不输出正向授信建议。
- 建议风险经理参与复核或进入人工专项复核。
- 现场核验问题应覆盖股权变更原因、诉讼进展、停工项目、真实债务负担、资金缺口和还款来源。
