# 信贷尽调规则口径

## 规则维度

| 维度 | 重点核查内容 |
|---|---|
| 基本信息与主体资质 | 成立时间、股权结构、法定代表人、经营异常、历史沿革 |
| 经营与现场核查 | 主营业务、订单真实性、人流货流、设备开工率、上下游集中度 |
| 财务与现金流 | 营收、利润、经营性现金流、负债率、应收账款、存货、合同资产 |
| 征信与公开风险 | 征信、授信、担保、涉诉、失信、行政处罚、负面舆情 |
| 关联方与担保圈 | 关联企业、关联交易、互保、共同债务、实控人负债 |
| 行业与区域 | 行业景气度、政策导向、区域集中度、业主/客户集中风险 |
| 授信用途与资金闭环 | 用途合理性、订单或项目匹配度、支付路径、回款监管条件 |

## 典型关注规则

### 财务与经营交叉验证

- 应收账款增速高于收入增速：需要核查大额应收真实性和账龄结构。
- 经营性现金流覆盖不足 1 倍：需要关注偿债来源稳定性和回款质量。
- 税票匹配度低于 90%：需要说明报表、开票与纳税差异原因。
- 合同资产或预付款异常增长：需要核查业务真实性和资金占用。

### 征信与公开风险

- 存在未结诉讼、被执行或频繁合同纠纷：作为高关注线索进入尽调问题清单。
- 对外担保规模较高：需要穿透被担保方经营和或有负债影响。
- 票据融资占用高且周转快：需要结合授信用途判断流动性压力。

### 关联方与担保圈

- 关联采购或销售占比较高：需要核验定价公允性和资金回流。
- 存在互保链条或共同债务：需要提升担保圈风险关注等级。
- 实控人个人负债明显上升：需要核验个人与企业资金边界。

### 行业与区域

- 建筑工程、供应链、贸易等回款周期长行业：要提高对订单、验收、物流、回款闭环的关注。
- 区域和客户集中度过高：需要关注单一区域或业主风险传导。

## 演示样例口径

- 宁波智能装备：中等关注，核心关注点为应收增长、客户集中、对外担保、税票差异。
- 成都供应链：重点关注，核心关注点为现金流承压、司法纠纷、关联采购、担保圈复杂。
- 武汉建筑工程：审慎介入，核心关注点为项目回款周期、区域集中、合同资产增长、回款监管。

## 证据纪律

每一条风险判断必须尽量对应明确证据类型：

- 财务报表或财务明细
- 银行流水或票据融资记录
- 税票与纳税信息
- 合同、订单、发票、物流、验收或回款凭证
- 征信、授信、担保信息
- 司法、行政处罚或公开舆情
- 关联企业、股权和实控人信息
