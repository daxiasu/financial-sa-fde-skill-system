# 信贷知识问答 Skill

本 Skill 面向银行信贷知识问答场景，重点解决“提问自然、回答受控、来源可见、边界清楚”的问题。

## 核心工作流

1. 识别问题类型、制度主题、知识范围和回答边界。
2. 根据模式选择检索策略：严格模式 / 标准模式 / 搜索模式。
3. 限定知识库范围，过滤不在授权范围内的文档。
4. 抽取证据片段和引用锚点。
5. 按模板生成答案，并明确回答限制与下一步动作。

## 合规边界

- 严格模式下只允许基于命中知识片段回答。
- 未命中文档时不得编造制度结论。
- 对流程性问题可做结构化归纳，但必须保留依据来源。
- 若问题超出当前知识库范围，应明确提示范围限制。
