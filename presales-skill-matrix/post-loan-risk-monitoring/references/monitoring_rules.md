# 贷后风险监测规则

## 监测目标

本规则用于演示贷后风险变化识别，输出“贷后风险关注建议”，不输出正式五级分类、审批结论或处置决定。

## 风险变化等级

| 等级 | 含义 | 典型处理 |
|---|---|---|
| 稳定 | 未发现明显风险上升 | 常规跟进 |
| 轻微上升 | 出现单项轻度异常或信息缺口 | 客户经理补充核实 |
| 明显上升 | 多项异常叠加，或单项重要异常 | 纳入重点观察并形成跟进计划 |
| 重大预警 | 还款异常、重大负面、现金流恶化等高强度信号叠加 | 立即复核并提交风险经理 |

## 评分规则

基础分为 0。按以下规则累计：

### 还款行为

- 逾期 1-7 天：+18
- 逾期 8-30 天：+35
- 逾期超过 30 天：+55
- 最近 3 个月出现展期或借新还旧：+20
- 还款资金来源解释不足：+12

### 财务变化

- 营收同比下降 15%-30%：+12
- 营收同比下降超过 30%：+25
- 经营性现金流为负：+18
- 资产负债率超过 75%：+15
- 应收账款周转明显放缓：+12
- 存货周转明显放缓：+10

### 公开风险

- 经营异常或行政处罚：+12
- 被执行、诉讼、重大负面舆情：+25
- 监管处罚、严重违法、重大安全事故：+35

### 关联关系

- 关联交易占比 10%-25%：+8
- 关联交易占比超过 25%：+18
- 关联方出现重大负面：+20
- 异常资金往来解释不足：+15

### 行业和区域

- 行业景气度下行：+8
- 区域风险事件影响：+8
- 高波动行业且企业抗风险能力弱：+10

## 等级映射

| 分数 | 风险变化等级 |
|---|---|
| 0-14 | 稳定 |
| 15-34 | 轻微上升 |
| 35-59 | 明显上升 |
| 60 及以上 | 重大预警 |

## 调整原则

1. 避免过度报高风险。若仅有轻微信息缺口，不应直接给出重大预警。
2. 还款异常优先级高于一般财务波动。
3. 公开重大负面和还款异常叠加时，应提高关注等级。
4. 对贸易类企业，应特别关注应收账款、存货、关联交易和交易真实性佐证。
5. 输出时必须说明触发规则和人工复核点。
