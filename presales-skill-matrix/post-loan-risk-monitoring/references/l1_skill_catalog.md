# L1 原子 Skill 目录

本文件定义“贷后风险监测报告生成 Skill”可调用的 L1 原子能力。演示环境中，L1 调用以可解释模拟方式展示；真实环境中，可替换为平台内真实 Skill、工具、数据接口或模型链路。

## L1-01 企业基础画像生成

用途：

- 获取企业名称、行业、区域、授信金额、授信品种、客户经理、存量风险等级。
- 为后续风险判断提供客户上下文。

典型输出：

- 企业一页画像
- 授信敞口摘要
- 所属行业和区域标签

## L1-02 还款行为监测

用途：

- 检查是否出现逾期、展期、借新还旧、还款资金来源异常。
- 识别短期流动性压力。

典型输出：

- 还款表现状态
- 逾期天数
- 还款来源异常说明

## L1-03 财务异常指标识别

用途：

- 识别收入下滑、现金流为负、负债率上升、应收账款异常、存货异常。
- 判断经营基本面变化。

典型输出：

- 异常财务指标
- 变化幅度
- 风险解释

## L1-04 公开风险线索检索

用途：

- 检索司法、舆情、监管处罚、经营异常、重大负面新闻。
- 为风险变化提供外部信号。

典型输出：

- 公开风险线索
- 来源类型
- 人工复核提示

## L1-05 关联关系与交易识别

用途：

- 识别关联交易占比上升、异常资金往来、关联主体负面风险。
- 判断风险是否通过关联方传导。

典型输出：

- 关联主体
- 关联交易变化
- 风险传导说明

## L1-06 风险变化归因

用途：

- 将财务、还款、舆情、关联关系等信号汇总为风险变化原因。
- 解释为什么某客户风险上升。

典型输出：

- 风险变化等级
- 主要归因
- 次要归因

## L1-07 贷后处置动作生成

用途：

- 根据风险变化和业务场景生成客户经理跟进动作。
- 包括走访、补充材料、追加核验、额度复核、纳入观察名单等。

典型输出：

- 客户经理动作清单
- 风险经理复核事项
- 所需补充材料
