银行域首版矩阵：用 L1 能力积木支撑 L2 场景 Skill，并预留 L3 高级编排。

Author: daxiasu

行业域

银行 Skill 矩阵

L1

原子 Skill

大家共用的底层能力，更多由技术团队和业务专家共同建设。

L2

场景 Skill

基于员工当前业务场景沉淀，适合售前演示、客户交流和资产复用。

可演示独立任务

授信走访前穿透式风险分析

面向银行对公信贷经理，输出走访前风险关注等级、风险线索和询问清单。
可演示L1 调用

贷后风险监测报告生成

将财务、舆情、还款、经营变化整合为贷后风险监测报告。
L3

高级编排 Skill

高级骨干员工创建的复杂任务能力，适合非标业务、专项任务和现场创新。

Skill 体系说明

L1 原子 Skill 公共基础能力，由技术团队和业务专家共同建设。 例如：企业画像、材料检查、财务异常、关联关系识别。
→
L2 场景 Skill 员工围绕当前业务场景沉淀，可演示、可复用、可交付。 例如：授信走访前风险分析、贷后风险监测报告。
→
L3 高级编排 Skill 面向非标业务、专项任务和现场创新，调度多个 L1/L2。 例如：信贷任务 Planner、小微贷专项尽调助手。
客户现场任务 发现真实业务痛点，但不带出客户数据。
L3 快速编排 先处理非标场景，验证是否有业务价值。
L2 场景沉淀 把多次验证后的流程固化为标准场景 Skill。
L1 能力抽象 把高频共性能力沉淀为公共原子 Skill。
售前矩阵复用 用模拟数据和通用规则形成可展示资产。
